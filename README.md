#Lemon Eligibility Checker API
A simple API to check the eligibility of clients for Lemon services

## Description

This API was developed using the awesome [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deploy

Configure your aws credentials in the credentials file, inside the .aws folder

```yaml
[default]
aws_access_key_id = YOUR_ACCESS_KEY
aws_secret_access_key = YOUR_SECRET_ACCESS_KEY
```

then run

```bash
yarn run deploy
```
